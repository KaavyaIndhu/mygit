# MyGit — A Git-Inspired Version Control System (Go)

MyGit is a minimal version control system written in Go that demonstrates how Git works internally.  
It implements core concepts such as content-addressable storage, commit graphs, and branch pointers.

The goal of this project is to understand the internal architecture of Git by building a simplified version from scratch.

---

## Features

- Repository initialization
- Content-addressable object storage using SHA-1 hashing
- Staging area (index)
- Commit snapshots
- Commit parent graph
- Branch creation
- Branch checkout
- Commit history traversal
